About HK Street Food 3D

This website is an interactive Web 3D application that showcases iconic Hong Kong foods through real-time 3D models.

What You Can Explore

Egg tart, egg waffle, and milk tea are all represented as dedicated model pages so you can inspect each item in detail.

The aim of this project is to combine cultural storytelling with modern web technologies, allowing users to explore familiar local favourites in a more engaging and visual way.
